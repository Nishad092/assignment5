Question-1: What are the different ways to select an element in the DOM?

getElementById('id'),getElementsByClassName('class'),querySelector(''),querySelectorAll('').

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML-sets the html content.
innerText-sets the text only.
textContent-sets the full text.

Question-3: What is event delegation in the DOM?

its a process where parent element handles events for child element.

Question-4: What is event bubbling in the DOM?

It is a mechanism where an event triggered on a child element propagates upward through its ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

-create element > document.createElement(),
-add element > body.appendChild()
-remove element > removeChild();
